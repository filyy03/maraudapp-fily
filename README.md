Maraud’App — Plateforme solidaire de gestion de maraudes

Maraud’App est une plateforme web conçue pour faciliter l’organisation des maraudes solidaires en centralisant la gestion des bénévoles, des dons et des distributions.

Elle permet aux associations de mieux structurer leurs actions terrain et d’optimiser la coordination des équipes.



--> Objectif

Créer un outil simple et efficace pour :
	•	organiser des maraudes
	•	mobiliser des bénévoles
	•	trier et gérer les dons selon les besoins réels
	•	piloter les actions via un tableau de bord centralisé



--> Technologies utilisées
	•	HTML / CSS — Structure et interface
	•	JavaScript — Interactions et logique front-end
	•	PHP — Back-end et traitement serveur
	•	SQL — Base de données
	•	IA — utilisée pour améliorer la clarté des commentaires, structurer la documentation et optimiser l’organisation du projet



--> Fonctionnalités
	•	Inscription et gestion des bénévoles
	•	Formulaire proposition de dons
	•	Système de rôles (distribution, logistique, coordination, etc.)
	•	Calendrier des maraudes
	Gestion des dons avec tri :
	•	Accepté
	•	En attente
	•	Refusé (selon les besoins)
	•	Tableau de bord administrateur

--> Évolutions possibles
	•	Notifications bénévoles
	•	Statistiques avancées
	•	Géolocalisation des maraudes
	•	Proposer différents rôles aux bénévoles (logistique, distribution, coordination, etc.)
	•	Permettre aux bénévoles de choisir leur rôle préféré
	•	Gestion automatique du stock
	•	Les dons récupérés sont automatiquement déplacés vers le stock disponible
